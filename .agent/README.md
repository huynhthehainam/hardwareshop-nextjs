# Shared AI Assets

This folder stores reusable, agent-agnostic assets for this repository.

## Structure

- `skills/`: shared skills usable by any coding agent
- `gemini-settings.json`: copied reference of Gemini MCP settings

## Compatibility

- `.gemini/skills` is a junction pointing to `.ai/skills`.
- If you add or edit skills, do it in `.ai/skills`.
